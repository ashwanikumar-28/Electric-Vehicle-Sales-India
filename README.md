## Electric Vehicle Sales by State in India

## Project Overview
This project analyzes Electric Vehicle (EV) sales across Indian states using Data Analysis and Machine Learning.

## Dataset
- File: Electric Vehicle Sales.csv
- Contains EV sales data by state, year, vehicle category and type

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- VS Code

## Machine Learning Model
- Algorithm: Random Forest Regressor
- Evaluation Metric: Mean Squared Error (MSE)
- Model MSE: 130.58
